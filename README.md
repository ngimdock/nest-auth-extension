# Nest-auth-extension

Master the passport, jwt, google authentication and role-based access control, claims-based and policy-based authorization with nestjs application.

## Start the app

### Install dependencies

```bash
yarn install
```

### Setup database

```bash
yarn db:dev:restart
```

### Run the app

```bash
yarn start:dev
```

### Navigate to

```bash
http://localhost:3333/status
```
